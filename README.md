# s9_parquet

Parquet file management based on Apache Arrow Parquet

https://crates.io/crates/parquet

`cargo install parquet`

Running the above command will globally install these binaries:
- parquet-concat
- parquet-fromcsv
- parquet-index
- parquet-layout
- parquet-read
- parquet-rewrite
- parquet-rowcount
- parquet-schema
- parquet-show-bloom-filter
